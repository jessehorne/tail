tail.py
=======

tail.py is a *nix, tail, clone. It is used for output the last lines or characters of a file.

How to use
----------
```
python tail.py OPTION INTEGER FILENAME

python tail.py -n 5 bin/test1.txt
python tail.py -c 10 bin/test2.txt
```
